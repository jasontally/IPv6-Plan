About This Tool

Hierarchically divide IPv6 address space for network planning and documentation.

Key Features:

  • Nibble-Aligned Splitting (Default): Click the green "Split" button to divide a network into smaller subnets at 4-bit boundaries (/20, /24, /28, etc.), creating 16 child subnets per split.
  • Custom Split Targets: Use the dropdown next to the Split button to select any target prefix for geographical denomination models or specialized addressing schemes.
  • Join Subnets: Click red "Join" button to merge subnets back together into their parent network.
  • Prefix Range: /16 to /64, with /64 as minimum assignable subnet.
  • Location-Based Planning: Shows count of /48s (typical site/location size) for prefixes < /48, and /64s (minimum subnet size) for prefixes â‰¥ /48.
  • Visual Hierarchy: Join buttons span multiple rows to show parent-child relationships.
  • Annotations: Add notes and color-code subnets for documentation.
  • Shareable URLs: Complete state encoded in URL for easy sharing.
